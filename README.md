# habit_tracker

# Setup
```bash
npm i fastify
npm i typescript -D
npx tsc init # create tsconfig.json file
# change tsconfig.target from es2016 to es2020
npm i tsx -D
# create dev script in package.json with tsx watch src/server.ts
```
